## Séance 11/05/2026

### BI (Business Intelligence) : 
Consiste à transformer les données brutes en informations exploitables pour soutenir la prise de décision et les activités opérationnelles au sein d'une organisation

### BI vs Data Science
#### BI :
- La BI se concentre sur la transformation des données brutes en informations exploitables pour soutenir la prise de décision et les activités opérationnelles
- La BI utilise principalement des outils de reporting, de requêtes et de visualisation pour analyser les données
- Les analyses de BI se basent souvent sur des données agrégées et structurées, et les modèles sont généralement définis à l'avance
#### Data Science :
- La Data Science vise à découvrir des informations précieuses à partir des données, à créer des modèles prédictifs et à automatiser la prise de décision
- La Data Science utilise des techniques avancées d'analyse de données, de statistiques et de machine learning

### Le DataSet (BI) : 
Une collection structurée de données, souvent sous forme de table, contenant des informations organisées en lignes et en colonnes

### Le DataSet (Data Science) : Un ensemble des exemples de données sous forme (X, y) :

- X : porte le nom de « feature / facteur » sous forme d’une matrice / plusieurs variables X = (x1, x2, x3, ….)
- y : porte le nom de « target / cible » : La valeur qu’on cherche à prédire à partir de X

### PowerBI :

- PowerBI Desktop : Une application de bureau gratuite pour la création de rapports et de tableaux de bord interactifs
- PowerBI Service : Une plateforme basée sur le cloud pour le partage, la collaboration et la publication des rapports créés avec Power BI Desktop

### Utilisation DataSet en BI (PowerBI) :
- Collecte et préparation des données (excel, csv, json, xml, sql, ...)
- Nettoyer les données, éliminer les doublons, traiter les valeurs manquantes, et formater les colonnes
- Importer le dataset dans Power BI en utilisant des connecteurs spécifiques
- Créer un modèle de données pour relier plusieurs datasets si nécessaire (relations entre différentes tables)
- Créer des colonnes calculées, des regroupements, ou des indicateurs clés
- Utiliser les données pour construire des tableaux de bord, des graphiques, et des KPI
